# Resume Builder

A modern, high-fidelity web application built with **Next.js** and **Tailwind CSS** that generates professional resumes following the official **Harvard University (MCS) format**.


## 🚀 Key Features

-   **Harvard Formatting Engine:** Strictly adheres to Ivy League standards (Times New Roman, 16pt name, 11pt headers, 10pt body).
-   **Live High-Fidelity Preview:** Watch your resume take shape in real-time with 1:1 document rendering.
-   **Dynamic Form Entry:** Easily add, remove, and reorder multiple education and experience entries.
-   **Optional Sections:** Toggleable "Leadership & Activities" section to customize your professional story.
-   **Direct PDF Download:** High-resolution PDF generation with precise layout preservation.
-   **Sample Data:** Instantly populate the builder with professional sample data to see the format in action.

## 🛠️ Tech Stack

-   **Framework:** Next.js 15+ (App Router)
-   **Language:** TypeScript
-   **Styling:** Tailwind CSS (App UI) & Vanilla CSS (Resume Template)
-   **PDF Generation:** html2canvas & jsPDF

## Set Up

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```
